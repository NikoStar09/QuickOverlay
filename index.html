🖼️ QuickOverlay

Este proyecto es un bookmarklet personalizado para superponer imágenes en cualquier sitio web.

🚀 ¿Cómo usarlo?

  1. Copia el código del bookmarklet desde bookmarklet.js
  2. Crea un marcador en tu navegador
  3. Pega el código en el campo URL
  4. Haz clic sobre cualquier página para activarlo
